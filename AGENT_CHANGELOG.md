# AGENT_CHANGELOG

## Дата
- 2026-02-12

## Что сделано

1. Полностью переработан юридический контур в директории `hitorilink`.
- Создана новая структура `hitorilink/legal/` для раздельного хранения всех обязательных документов.
- Добавлен отдельный индекс юридических документов для централизованной навигации.
- Добавлена единая визуальная система оформления для всех документов через общий CSS.

2. Создана главная страница юридической навигации `hitorilink/index.html`.
- Страница теперь содержит перечень всех юридических документов с описаниями.
- Зафиксировано позиционирование сервиса как бота для знакомств по интересам (не дейтинг-приложение).
- Добавлено явное ограничение по возрасту 18+.
- В реквизитной части использованы данные оператора в формате ИП на НПД и плейсхолдеры `<PLACEHOLDER>`.

3. Созданы отдельные юридические документы в `hitorilink/legal/`:
- `public-offer/index.html`: публичная оферта с условиями услуг, оплаты, моментом оказания услуги, блокировками, изменением условий и ограничением ответственности.
- `privacy-policy/index.html`: политика обработки ПДн с перечнем данных, целями, основаниями, сроками, удалением, трансграничной передачей и правами пользователя.
- `user-agreement/index.html`: пользовательские правила, 18+, запреты, согласие на модерацию/ML-фильтрацию, автоограничение при 10 репортах, отсутствие гарантий результата.
- `refund-policy/index.html`: закреплено, что возврат проводится через оператора и только если оплата прошла, но бот не зафиксировал услугу.
- `moderation-policy/index.html`: запрещенный контент, механизм репортов, санкции, обжалование и основания блокировок.
- `age-policy/index.html`: отдельная возрастная декларация 18+ и закрепление ответственности пользователя за ложные сведения о возрасте.
- `payment-and-credits/index.html`: разъяснено, что внутренний баланс является невыводимыми кредитами предоплаты, а не электронными деньгами; добавлены антифрод/AML-положения.

4. Обновлен legacy-раздел `hitorilink/policy/`.
- `hitorilink/policy/index.html` изменен на страницу перенаправления к новой политике ПДн.
- `hitorilink/policy/policy.css` оставлен как совместимый legacy-файл и привязан к новому общему стилю.

5. Добавлены служебные документы по вашему запросу.

- Создан `AGENT_CHANGELOG.md` с детальной фиксацией всех выполненных изменений.

6. Приведена система плейсхолдеров к единому формату.
