# AGENT_LINKS

Ниже приведены все актуальные ссылки, созданные/обновленные в рамках юридического комплекта HitoriLink.

| Ссылка | Назначение |
| --- | --- |
| `/hitorilink/` | Главная юридическая навигация проекта HitoriLink. |
| `/hitorilink/legal/` | Единый индекс всех юридических документов. |
| `/hitorilink/legal/public-offer/` | Публичная оферта: условия услуг, оплаты, ограничения ответственности, блокировки. |
| `/hitorilink/legal/privacy-policy/` | Политика обработки персональных данных по 152-ФЗ. |
| `/hitorilink/legal/user-agreement/` | Пользовательское соглашение: правила использования, 18+, запреты и санкции. |
| `/hitorilink/legal/refund-policy/` | Политика возвратов: возврат только при подтвержденной техошибке. |
| `/hitorilink/legal/moderation-policy/` | Политика модерации: репорты, автоограничения, обжалование. |
| `/hitorilink/legal/age-policy/` | Возрастная декларация и подтверждение 18+. |
| `/hitorilink/legal/payment-and-credits/` | Условия платежей, внутреннего баланса (кредитов), anti-fraud/AML-подход. |
| `/hitorilink/policy/` | Legacy-ссылка: перенаправляет на актуальную политику ПДн. |

Дополнительно:
- Общий стиль документов: `/hitorilink/legal/legal.css`.
